# 문체 규칙

이 파일이 문체 규칙의 **단일 원본**이다. `agent-workspace/AGENT_RULE.md`, `agent-workspace/AGENT_RULE_SEED.md`, 폴더별 `AGENT_RULE.md` 등 다른 위치에서는 본문을 중복해서 적지 말고 이 파일을 링크로 참조한다.

- 콜론(`:`)은 레이블과 필드명 뒤에만 쓴다 (예 `성공 기준:`, `Sources`). 문장이나 구 끝, 도입구에는 콜론 대신 공백이나 쉼표를 쓴다 (예 "정리하면,").
- 인라인 나열 시 가운뎃점(`·`, `•`) 대신 쉼표를 쓴다 (예 "A, B, C").
